How To Use live_help

The main goal of the system is to evaluate the time complexity of the code obtained as an answer to the
selected question and to examine whether it fulfills the required complexity.

In order to achieve this, what you have to do is :

  • Pick a question from the list of questions.
  • Implement the answer to the question within the code editor panel, in such a way that satisfies the API provided
    in the panel, and without any class members (without any use of variables declared outside a method scope).
  • Check whether the code is compiled successfully by clicking 'Build Code' button.
  • After the code has been compiled, send the code to complexity assessment by clicking the 'Submit Code' button.
